# uae-real-estate-market-analysis-sql
## 🗄️ Database Architecture & Production Data
The analytical environment processes an optimized production-ready dataset containing **200 verified property transactions** distributed symmetrically:
1. `dubai_properties`: 100 rows capturing hyper-targeted luxury markets (Dubai Marina, Downtown Dubai, Palm Jumeirah).
2. `abudhabi_properties`: 100 rows capturing premium localized markets (Yas Island, Saadiyat Island, Al Reem Island).

*The full structural build and payload data can be found in `data_setup.sql`, and production queries are available in `queries.sql`.*
